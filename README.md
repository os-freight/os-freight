# OS-Freight Website v2

Statische, responsive Unternehmenswebsite für OS-Freight Solutions.

## Lokal testen
Öffne `index.html` direkt oder starte einen lokalen Webserver:

```bash
python3 -m http.server 8080
```

Danach: `http://localhost:8080`
